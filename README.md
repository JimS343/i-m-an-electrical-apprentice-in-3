# Apprentice Electrical Hub

This is a beginner-friendly website project made with plain HTML, CSS, and JavaScript.

## What each file does

- `index.html`: the structure and content of the website
- `style.css`: the design, colours, layout, and spacing
- `script.js`: the small interactive features like filtering and searching cards

## How to open the site

1. In this folder, double-click `index.html`.
2. Your web browser should open the website.

You do not need to install anything for this version.

## How to edit the site

1. Open the folder in a code editor like Visual Studio Code.
2. Change text in `index.html` to update headings, paragraphs, and topic cards.
3. Change colours and layout in `style.css`.
4. Change filter behaviour in `script.js` if you want more interactivity later.

## Good first edits to make

1. Replace the sample topic cards with notes from your current level of school.
2. Add more cards for topics like transformers, conduit fill, load calculations, motors, and safety.
3. Create a new section for formulas or exam prep questions.
4. Add links to trusted resources you use often.

## Important note

Use this site as a study tool. Always verify code rules and job requirements with the current Canadian Electrical Code, provincial requirements, manufacturer instructions, and your instructors or supervisors.
