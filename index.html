Electrical theory

Start with the fundamentals

A strong theory section helps everything else make sense. This is where you can explain current, voltage, resistance, power, magnetism, transformers, and basic troubleshooting steps.

Good habit: for each topic, add a definition, a formula if there is one, and one practical example from class or the field.

Residential wiring

Organize the work you see most often

This section is great for layouts, device boxes, cable runs, panels, service notes, and common troubleshooting situations in homes.

You can later break this into subtopics like kitchens, bathrooms, smoke alarms, arc-fault protection, and service upgrades.

Commercial wiring

Capture methods, materials, and systems

Use this section for conduit systems, box installation, support rules, lighting controls, motor basics, and the equipment that tends to show up outside residential jobs.

As you advance, you can add sketches, bend references, and common installation details.

Code and standards

Keep notes simple and clearly labeled

Code-related notes are most useful when they point you back to the official source. Write your own summaries, but include the section or table you learned from so you can verify it later.

Because code can change by edition and local adoption, date your notes and avoid presenting memory aids as final authority.
